```typescript

  /api â†’ http://localhost:3000/api
```

4. Success message â†’ CB auto-refreshed â†’ Banking history updated

### Pooling Flow:
1. Set Year â†’ Existing pools auto-loaded from `/api/pools?year=X`
2. Add members â†’ Enter Ship IDs + CB values
3. Click "Fetch CB" for each member â†’ Optional auto-fill
4. Verify pool validation â†’ Click "Create Pool" â†’ API call to `POST /api/pools`
5. View pool result â†’ Shows before/after CB for all members
6. Existing pools list refreshes automatically
